Last resort

Soms is je computer niet meer te redden. Of tenminste de hoeveelheid tijd die er in gestoken moet worden om alle malware en virussen te verwijderen is gewoon te groot. In deze gevallen is het best om terug te grijpen naar je laatste back-up en je hele systeem gewoon opnieuw te installeren.

Een waarschuwing. Als je niet veel van computers kent is het beter om nu een persoon die iets van computers af weet het over te laten nemen. In sommige instructies die volgen neem ik aan dat je vertrouwd bent met meer geavanceerdere procedures.

Om dit uit te voeren heb je eerst je gewenste versie van windows (of een ander os indien gewenst.) nodig op verwisselbare media. Meestal zijn installatie cd’s meegeleverd bij aankoop van een computer. Maar indien deze niet bij de hand zijn kan me de iso’s bij microsoft downloaden. De tool van Microsoft kan zowel direct op usb installatie media aanmaken of een iso indien dit gewenst is.

Aanmaken van installatiemedia

Indien je geen installatie cd's hebt dan zul je eerst installatie media moeten aanmaken.

Windows 10:

Download de windows 10 tool via deze link en voer de executable uit. De setup zal je door het process om een bootable usb stick te loodsen. Zoals eerder aangehaald, kan deze tool ook een iso file maken indien gewenst. In deze gids nemen we aan dat je direct een usb bootable hebt aangemaakt met deze tool.

windows 8.1/7/ lager of een linux systeem.

Voor andere OS'en dan windows 10 zijn er geen tools om direct een usb bootable stick aan te maken zoals met de vorige tool. Dus nu hebben we zowel een iso image nodig van het gewenste OS. Meestal zijn deze te vinden op de site van Microsoft voor windows (vereist een geldige windows key.) of op de site van de linux distributie.

Dan hebben we ook een tool nodig om van deze iso een bootable usb stick te maken. Hiervoor gebruik ik meestal rufus. Dit is een simpele tool die men op de site van rufus kan vinden. Run de executable die je daar download. En selecteer de gewenste USB en ISO zoals aangegeven in het gegeven voorbeeld. Klik daarna op start en wacht tot het proces gedaan.

Installatie van OS

Bij windows 7 of lager en linux machines is het starten van de installatie procedure simpel. Sluit de installatiemedia aan de computer en start de PC op. Er zou een scherm moeten tevoorschijn komen met de bootschap. “press [toets] to choose boot device.” Druk op die knop en wacht even. Er zou na een tijd een lijst moeten tevoorschijn komen met een lijst van devices waarvan je kan boeten. Selecteer je installatie media en de setup zou moeten starten.

Windows 10 en 8 gebruiken UEFI in plaats van BIOS en men kan niet zomaar van een usb stick booten. Begin met de installatiemedia aan de computer te koppelen en start de pc op. Wacht tot men in het login scherm komt en houd dan de shift toets ingedrukt terwijl men op de restart knop drukt. In het scherm dat volgt klik op advanced startup options. Daarna op de optie use a device en selecteer je de usb stick waarvan je wil opstarten.
